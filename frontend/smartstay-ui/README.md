# SmartStay UI

Frontend for SmartStay — a full-stack hotel booking and management platform.

## Project Overview
SmartStay is a real-world hotel booking system built for showcasing backend architecture, booking workflows, and frontend product design.

### Tech Stack
- Angular (Standalone)
- Bootstrap
- TypeScript
- REST API integration (ASP.NET Core backend)

### Current Status
- Home page UI (Landing page) implemented
- Navbar, hero section, search box, featured hotels, offers, and footer designed
- Responsive layout applied using Bootstrap
- Static UI ready, API integration planned next
